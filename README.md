## 💼 Prerequisites

* Node.js
* npm

## ⚙️ Installing

* Clone the latest code on the master branch
* Install dependencies
    * ```npm install```
* Run the web server!
    * ```npm run dev```

## 🎈 Usage

* Navigate to http://localhost:3000 on your browser
